API доступ к данным Windqix

Программный доступ к данным аналитики поведения пользователей для интеграции с вашими системами

Что такое API Windqix?

Гибкие возможности интеграции данных аналитики с вашей инфраструктурой

API Windqix предоставляет программный доступ к данным поведенческой аналитики, позволяя интегрировать их с вашими системами и автоматизировать процессы. С помощью API вы можете получать все типы данных, доступных в интерфейсе Windqix, и использовать их в своих приложениях, дашбордах и отчетах.

С помощью API Windqix вы сможете:

  • Интегрировать данные о поведении пользователей в ваши существующие системы
  • Создавать собственные дашборды и отчеты с данными Windqix
  • Автоматизировать процессы анализа и оптимизации конверсии
  • Разрабатывать собственные приложения с использованием данных поведенческой аналитики

API Windqix использует современный REST-интерфейс, поддерживает аутентификацию по ключам API и возвращает данные в формате JSON, что делает его совместимым с большинством современных систем и языков программирования.

Возможности API Windqix

Полный доступ к данным аналитики через программный интерфейс

Доступ к данным аналитики

Получайте программный доступ ко всем данным аналитики поведения: тепловым картам, сессиям, воронкам и другим типам данных.

Мощная фильтрация

Используйте гибкие параметры фильтрации для получения только тех данных, которые вам нужны, включая временные диапазоны, сегменты и метрики.

Вебхуки и колбэки

Настраивайте автоматические уведомления о новых данных и событиях через вебхуки и колбэки для реагирования в реальном времени.

SDK для популярных языков

Используйте готовые SDK для Python, JavaScript, PHP, Java и других языков, чтобы ускорить интеграцию с API Windqix.

Безопасный доступ

API Windqix обеспечивает безопасный доступ к данным через HTTPS, используя токены API и гранулированные права доступа.

Высокая производительность

API оптимизирован для быстрого отклика и обработки больших объемов данных, поддерживает пагинацию и потоковую передачу.

Пример использования API

Простой пример получения данных о конверсии через API Windqix


// Пример GET-запроса к API для получения данных о конверсии
const url = 'https://api.windqix.ru/v1/analytics/conversion';
const params = new URLSearchParams({
  start_date: '2023-01-01',
  end_date: '2023-01-31',
  segment: 'mobile_users',
  page: '/checkout'
});

// Добавляем ключ API в заголовки запроса
const headers = {
  'Authorization': 'Bearer YOUR_API_KEY',
  'Content-Type': 'application/json'
};

// Выполняем запрос
fetch(`${url}?${params}`, { headers })
  .then(response => response.json())
  .then(data => {
    console.log('Данные о конверсии:', data);
    // Обработка полученных данных
  })
  .catch(error => console.error('Ошибка при запросе к API:', error));

Часто задаваемые вопросы

Ответы на популярные вопросы об API Windqix

Какие лимиты на API-запросы предусмотрены в разных тарифных планах?

Лимиты на API-запросы зависят от вашего тарифного плана. Тариф "Стартап" включает до 10 000 запросов в день, "Бизнес" - до 100 000 запросов в день, "Энтерпрайз" - до 1 000 000 запросов в день. Если вашему бизнесу требуется больше запросов, мы можем предложить индивидуальные решения. Также доступна опция покупки дополнительных квот API-запросов для любого тарифа.

Насколько безопасен доступ к данным через API?

API Windqix использует современные стандарты безопасности. Все запросы передаются через HTTPS с TLS 1.2 или выше, для авторизации используются JWT-токены с ограниченным сроком действия, доступны двухфакторная аутентификация и ограничение по IP-адресам. Также API поддерживает гранулированные права доступа, позволяя ограничивать доступ к определенным типам данных и операциям для разных ключей API.

Для каких языков программирования доступны SDK?

Windqix предоставляет официальные SDK для следующих языков и платформ: JavaScript (Node.js и браузер), Python, PHP, Java, Ruby, Go, .NET (C#), Swift (iOS) и Kotlin (Android). Эти SDK значительно упрощают интеграцию с API, предоставляя типизированные интерфейсы, обработку ошибок и другие полезные функции. Для других языков и платформ доступна подробная документация по работе напрямую с REST API.

Какие форматы экспорта данных поддерживаются?

API Windqix позволяет получать данные в нескольких форматах: JSON (основной формат), CSV (для табличных данных), XML (для интеграции с устаревшими системами) и PDF (для готовых отчетов). Для некоторых типов данных, таких как тепловые карты, доступен экспорт в форматах изображений (PNG, JPEG) или SVG. Для больших объемов данных поддерживается потоковая передача и возможность постраничного получения результатов.
